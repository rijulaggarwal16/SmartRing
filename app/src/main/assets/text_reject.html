Text On Auto Call Reject

This feature allows you to send a text message as a reply to the caller, whose call has been rejected by 'No Call Silent' mode.

Enable/Disable Feature:

You can enable or disable this feature from the main preferences menu. This feature however works only when smart ring rejects a call, and can be used along with other features as well.
Note: This feature only acts when incoming calls are rejected due to 'No Call Silent' mode of this app. It is not a substitute for a permanent reply by text on call reject feature provided by other apps.

Set message of the auto reply text:

To change the message sent when this feature is enabled, just click on the row in the preferences screen. There is still a default message that will be used if not set.
